# excel-dashboard
This repository showcases my end-to-end data analysis workflow using *Excel, Power BI, and Tableau*. It features a deep dive into E-commerce and Food Delivery (Swiggy) data.
1. Power BI: E-Commerce Sales Dashboard
*File:* ECOMMERCE SALES DASHBOARD.pbix
* *Objective:* My first Power BI project focusing on automated data modeling and interactive storytelling.
* *Key Features:* Used DAX measures to calculate YTD sales and profit margins. 
* *Visuals:* Dynamic slicers for category-wise filtering and trend forecasting.
 2. Excel: Interactive Swiggy Dashboard
*File:* MY SWIGGY DASHBOARD.xlsx
* *Objective:* Analyzing Swiggy's sales performance and customer ordering patterns.
* *Technical Skills:* Advanced Pivot Tab* Slicers, and *Custom Number Formatting* (e.g., Millions formatting).
* *Insights:* State-wise sales distribution using Map Charts and Food Type (Veg/Non-Veg) analysis.
 3. Tableau: Visual Exploration
*File:* my board.twb
* *Objective:* Exploring advanced data visualization techniques in Tableau.
* *Key Focus:* Creating high-impact geographical maps and weekly sales trend chart
